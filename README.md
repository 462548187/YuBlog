# YuBlog

## 介绍

功能强大的个人博客，功能齐全的管理后台，简洁大气的前端页面，集成富文本编辑器支持`markdown`语法，集成畅言评论。

部署方案查看：[Ubuntu+uwsgi+Nginx部署Flask应用](http://www.yukunweb.com/2017/12/ubuntu-nginx-uwsgi-flask-app/)


## 其他

- 个人使用博客，计划长期更新改进；
- 欢迎`pull requests`，提供建议；
- 如果喜欢欢迎`star`和`fork`。
